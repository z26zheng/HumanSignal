# HumanSignal

HumanSignal is a Chrome extension for local signal intelligence on LinkedIn.

The repository currently contains:

- `human-signal/`: WXT extension implementation.
- `Plan/`: execution plans and integration gates.
- `Scoping/`: product and architecture documents.
- `BestPractice/`: engineering standards for the codebase.

## Development

```bash
cd human-signal
corepack pnpm install
corepack pnpm dev
```

## Verification

```bash
cd human-signal
corepack pnpm compile
corepack pnpm test
corepack pnpm build
```
